# Chapter 1: Foundation Algebra Review

## Overview

Engineering Mathematics begins by reviewing foundational algebra. Many of the skills used in this chapter are foundational mathematical tools that you will need to keep using repeatedly both in this course and beyond.

## Key Concepts

### Order of Operations (BEDMAS)

When evaluating a mathematical expression, the order of operations is important. The acronym **BEDMAS** is used for simplifying expressions:

- **B**rackets
- **E**xponents
- **D**ivision
- **M**ultiplication
- **A**ddition
- **S**ubtraction

This means you must resolve any brackets before evaluating exponents, e.g., $(3-1)^2 = 2^2 = 4$, and handle multiplication before addition, e.g., $3 \times 4 + 2 = 12 + 2 = 14$.

### Solving Equations

The *reverse* is true for solving equations. First you must "undo" any subtraction or addition to isolate the variable.

**General Rule:** You can do any mathematical operation to the equation as long as you do it to both sides.

**Example 1:** Solve for $x$: $x - 11 = 7$

Add 11 to both sides: $x - 11 + 11 = 7 + 11$

Simplify: $x = 18$

**Example 2:** Solve: $2x + 5 = 10$

Subtract 5 from both sides: $2x = 5$

Divide both sides by 2: $x = \frac{5}{2}$

### Rearranging Equations

Rearranging equations is a handy tool in analysis to see how variables influence relationships. The equation for a straight line is $y = mx + c$ with two variables $x$ and $y$. The equation is arranged so that variable $y$ is alone on the left-hand side, making $y$ the subject.

**Steps to rearrange:**
1. Identify operations on the new subject
2. Undo operations in reverse order of BEDMAS

**Example:** Make $v$ (velocity) the subject in kinetic energy: $E_k = \frac{1}{2}mv^2$

1. Multiply both sides by 2: $2E_k = mv^2$
2. Divide both sides by $m$: $\frac{2E_k}{m} = v^2$
3. Square root both sides: $v = \sqrt{\frac{2E_k}{m}}$

## Formulas

### Quadratic Formula
For a quadratic equation $ax^2 + bx + c = 0$:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

### Difference of Squares
$$a^2 - b^2 = (a + b)(a - b)$$

### Perfect Square
$$(a + b)^2 = a^2 + 2ab + b^2$$

$$(a - b)^2 = a^2 - 2ab + b^2$$

### Straight Line
$$y = mx + c$$

where $m$ is the slope and $c$ is the y-intercept.

## Learning Objectives

1. Apply the order of operations (BEDMAS) correctly
2. Solve linear equations for unknown variables
3. Rearrange formulas to make different variables the subject
4. Factor quadratic expressions
5. Apply the quadratic formula to solve equations
6. Simplify algebraic expressions
